## Install

For linux:

```sh
./install
```

For windows:

```powershell
./install.ps1
```

## Fix missing icons/emojis and alignment issues

Install fonts:

```sh
pacman -S noto-fonts-emoji noto-fonts-cjk
```

Install symbols nerd font:

```sh
pacman -S ttf-nerd-fonts-symbols
```
